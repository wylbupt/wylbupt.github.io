WordPress, Hexo, Jekyll网站建设

发表于 2019-10-28

建设自己的网站，尝试一下吧！

常见的网站搭建方案（不分先后）

1）WordPress用PHP语言写的，
2）Hugo使用go语言是一种编译型语言，速度非常快，
3）Jekyll使用ruby编写，
4）Hexo使用nodejs编写，
ruby与nodejs都是面向对象的高级解释型语言，执行效率比编译型语言要低。（考验耐心……）

推荐以下2种

WordPress + VPS + 域名（收费）

WordPress 能够三分天下占其一，还是很流行的；
VPS的话可以使用谷歌云或其它，有免费或付费；
域名的话可以免费在freenom网站申请，当然资金充足的话也可以购买.com。

有了域名和VPS后，将域名绑定到VPS的IP地址，VPS安装宝塔面板后可以方便地搭建网站。
推荐2个不错的主题：hestia和neve
推荐1个不错的插件：elementor

Git+Github+Markdown+Hexo （免费）

1）安装node.js
2）安装https://hexo.io/zh-cn/，推荐next主题
3）部署网站到本地上
npm install hexo-cli -g
hexo init blog
cd blog
npm install
hexo server
4）部署网站到git.oi上
hexo server
hexo generate
hexo deploy

5）文本编辑推荐VScode免安装版+Markdown插件。

写博客首选Hexo，中小型网站建议WordPress，企业级网站肯定Java啦！
